# 📸 BoothStory

Aplikasi untuk menyimpan kenangan fotobooth dengan teman-teman.

## ✨ Fitur
- Upload foto & video
- Kelompokkan berdasarkan tanggal
- Download media
- Desain biru-putih aesthetic

## 🛠️ Tech Stack
- **Frontend**: Next.js 14, Tailwind CSS
- **Backend**: Express.js
- **Database**: MySQL
- **Storage**: Local (multer)

## 🚀 Cara Menjalankan

### 1. Clone repo
\`\`\`bash
git clone https://github.com/raniinyourheart/boothstory.git
cd boothstory
\`\`\`

### 2. Setup Backend
\`\`\`bash
cd backend
npm install
cp .env.example .env
# Edit .env dengan konfigurasi database kamu
npm run dev
\`\`\`

### 3. Setup Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

### 4. Buka browser
\`\`\`
http://localhost:3000
\`\`\`
